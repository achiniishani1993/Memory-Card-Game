# Memory-Card-Game
Build a simple memory matching game where users click cards to find matching pairs.

A simple and interactive Memory Card Game built using JavaScript, HTML, and CSS. The goal of the game is to match all pairs of cards with the least number of attempts.

# Features
* Shuffle cards randomly on each game start
* Click to flip cards and reveal symbols
* Match two cards to score points
* Unmatched cards flip back automatically
* Tracks score dynamically
* Displays messages for match, mismatch, and game completion
* Winning message when all pairs are matched 

# Technologies Used
* HTML
* CSS
* JavaScript (Vanilla JS)

# How to Play
* Click on any card to reveal its symbol
* Click on a second card to find a match
   * If the cards match:
         * They remain visible
         * Your score increases
   * If the cards do not match:
         * They flip back after a short delay
* Continue until all pairs are matched
